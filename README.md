# Global badges Replugged
Shows users awesome profile badges on their profile page.

## Features
- Aliucord Badges
- Better Discord Badge
- GooseMod Badges
- Enmity badges
- Custom Badges DB (get the badges here: [Discord](https://discord.gg/eTvYv95PCG "My Official Server"))
